This folder contains all the source code files required to complete Assignment 3.
This project focuses mainly on a single topic: the Breakout game. The basic implementation of the game is included in
Breakout.java, with extended functionality added in BreakoutExtra.java. The original version of the game was invented
by Steve Wozniak and was a staple included application for Apple devices. This project is a demonstration of intermediate
Java implementation, including global variables, mouse event listeners, animation, sound effects, and more.
