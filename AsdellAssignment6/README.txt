This folder contains all the necessary components required to complete Assignment 6: NameSurfer.
The VoteCounter class is a warmup exercise related to array manipulation.
The NameSurfer program is a self-contained statistical analysis program designed to graphically display name popularities as given by the names-data.txt file.
The NameSurferExtra program is a extension of the original program written to increase the functionality and readability of the graphical display.
